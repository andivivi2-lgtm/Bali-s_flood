\# Bali Flood and Extreme Rainfall Analysis



This repository contains MATLAB scripts for analyzing extreme rainfall and flood dynamics in Bali.



\## Repository Structure



\- MATLAB scripts

\- Small supporting CSV files



\## Large Dataset (Stored Externally)



The large NetCDF and supporting datasets (≈2.8 GB) are stored externally due to GitHub file size limitations.



📥 Download the full dataset here:

https://drive.google.com/drive/folders/1n2JWu4rG8wXxM0Ojo0aTcgBIa79trf0T?usp=sharing



After downloading, place the data files inside the project directory before running the scripts.
